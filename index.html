Projetos Realizados

Monitoramento de caixa d'água

Desenvolvimento e implementação de um sistema de monitoramento inteligente para caixas d'água utilizando tecnologia IoT. O projeto envolveu o uso de um ESP32 integrado a sensores de nível e pressão para medir a quantidade de água em tempo real. Os dados coletados são transmitidos para uma interface web responsiva, desenvolvida com HTML e Bootstrap, permitindo que o usuário visualize o nível da água e configure parâmetros personalizados. O sistema também inclui alertas configuráveis para situações de abastecimento e consumo, otimizando o gerenciamento de recursos hídricos.

Dosadora produção de bactérias

Desenvolvimento e automação de um sistema dosador para a produção de bactérias utilizadas em Estações de Tratamento de Efluentes (ETE) e aplicações agrícolas no combate a pragas. O projeto foi projetado para garantir precisão no controle de dosagem e condições ideais para o cultivo das bactérias, otimizando sua eficácia nos processos de tratamento de resíduos e no manejo agrícola sustentável. A solução inclui sensores para monitoramento de variáveis críticas, automação do fluxo de dosagem e integração com controladores para assegurar eficiência operacional e confiabilidade nos resultados.

Painel de comando

Painel de comando projetado para acionamento seguro e eficiente de queimadores. Desenvolvido com componentes de alta qualidade, proporcionando controle preciso, proteção contra falhas e operação automatizada, garantindo eficiência energética e segurança no processo.

Display lcd

Monitoramento em tempo real do nível da caixa d'água com exibição precisa em um display LCD. A leitura em porcentagem proporciona controle eficiente e prático, facilitando o acompanhamento e a gestão do consumo de água.

Balança rodoviária

Desenvolvimento de um sistema de integração IoT para balança rodoviária, permitindo a comunicação entre a balança e o sistema do computador via Wi-Fi. O projeto foi projetado para otimizar o monitoramento e a gestão dos dados de pesagem, eliminando a necessidade de conexões físicas e garantindo maior eficiência e praticidade no processo.

Instalação de motores

Realizada a instalação de motores em um sistema de pirólise, assegurando a funcionalidade e eficiência operacional do equipamento. O trabalho incluiu a configuração elétrica e mecânica dos motores, garantindo conformidade com as especificações técnicas e os requisitos do processo industrial.
